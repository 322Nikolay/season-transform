# Season Transform (Image Processing Project)

Проект по обработке изображений с использованием Python и OpenCV.

## 📌 Описание

Реализован алгоритм изменения сезона на фотографии:

- Осень → Лето
- Лето → Осень

Изменение выполняется за счёт анализа цветового пространства HSV и корректировки оттенков листвы.

---

## ⚙ Используемые технологии

- Python 3
- OpenCV
- NumPy

---

## 🧠 Алгоритм работы

1. Загрузка изображения
2. Перевод изображения из BGR в HSV
3. Создание маски по диапазону оттенков (Hue)
4. Изменение цветовых параметров:
   - Hue (оттенок)
   - Saturation (насыщенность)
   - Value (яркость)
5. Обратное преобразование в BGR
6. Сохранение результата
